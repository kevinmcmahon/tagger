Class Hierarchy

  • tagger.Rater: Class for estimating the relevance of tags
    • extras.NaiveRater: Rater subclass that jusk ranks single-word tags by their frequency and weight
  • tagger.Reader: Class for parsing a string of text to obtain tags
    • extras.SimpleReader: Reader subclass that doesn't perform any advanced analysis of the text
    • extras.UnicodeReader: Reader subclass that converts Unicode strings to a close ASCII representation
      • extras.HTMLReader: Reader subclass that can parse HTML code from the input
  • tagger.Stemmer: Class for extracting the stem of a word
    • extras.FastStemmer: Stemmer subclass that uses a much faster, but less correct algorithm
  • tagger.Tag: General class for tags (small units of text)
    • tagger.MultiTag: Class for aggregates of tags (usually next to each other in the document)
  • tagger.Tagger: Master class for tagging text documents
